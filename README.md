# OrderMate

> 聚焦于用户需求的点餐助手，提供优质的服务和体验

## 商家前台

- 登陆
- 叫号
- 订单
    - 出单
    - 结算
- 人工客服/智能客服

## 商家后台

- 登陆
- 录入餐桌
- 营业额统计
- 小程序布局调整
- 会员机制（会员等级｜折扣）
- 活动机制（优惠券｜banner）
- 分店机制
- 授权管理

## 客户端

- 游客/登陆
    - 多端信息同步
- 菜单展示
- 点餐功能
    - 支持多人共点
    - 结算
- 订单管理
    - 登陆后可以绑定游客订单
    - 微信小程序可以使用 wx.getStorageSync 将数据缓存在本地 
- 评价功能
- 客服服务
